## 🤖 Gemini Edition の特徴

### 🔄 OpenAI → Gemini 移行内容
- API Provider: OpenAI → Google Gemini Pro
- Model: GPT-4 → gemini-pro
- API Key: OPENAI_API_KEY → GEMINI_API_KEY
- Endpoint: OpenAI Chat Completions → Gemini generateContent
- Response Format: OpenAI互換 → Gemini Native

2. **新エージェント**
   - 日本語スペシャリスト - 日本語特化対話

### 📡 API エンドポイント
- `/api/chat` - Gemini Pro会話API
- `/api/agents` - エージェント一覧（Gemini対応）
- `/api/health` - システム状態（Provider情報含む）
- `/api/auth/login` - JWT認証

### 🛠️ 技術スタック
- **Backend**: PowerShell 7.0+ (Gemini API統合)
- **Frontend**: Vanilla JavaScript (Gemini特化)
- **AI Provider**: Google Gemini Pro
- **Authentication**: JWT Token
- **Security**: CORS, Rate Limiting, Input Validation
- **Deployment**: Azure App Service, GitHub Actions

### 🚀 起動方法
1. Google AI Studio でAPIキー取得
2. start-server.bat でGEMINI_API_KEY設定
3. サーバー起動: start-server.bat
4. フロントエンド: frontend/index.html

### 🔗 重要なURL
- Google AI Studio: https://makersuite.google.com/app/apikey
- Gemini API Docs: https://ai.google.dev/
- GitHub Repository: https://github.com/chnmotoTmz/mcp-agent-system
